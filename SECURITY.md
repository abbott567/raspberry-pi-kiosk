# Security Policy

## Supported versions

This project is a set of setup scripts and a guide rather than a versioned application. Only the latest version on the `main` branch is supported.

## Reporting a vulnerability

If you find a security issue, for example, a script that exposes credentials, weakens the Pi's configuration, or downloads something unsafely, please do not open a public issue.

Instead, report it privately using GitHub's private vulnerability reporting for this repository: <https://github.com/abbott567/raspberry-pi-kiosk/security/advisories/new>

You should get an acknowledgement within a few days. Once the issue is confirmed and fixed, it will be noted in the changelog, and you will be credited if you would like to be.
